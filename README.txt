PanYiBo专用Git版本--项目管理仓库
主要包含：开关电源、车载驱动OBC、集成电路、电机驱动、电子负载类仪器仪表等相关项目
区分规则：
    大类：以产品类型来进行区分。
    子类：以产品使用的MCU类型来进行区分，同时开发DSP、STM32、车载MCU等处理器。
    子类1:会有对应的仿真模型进行存放，那么就是以开发的工具来进行命名。
    
    （注意：没有进行子类分类的是以纯硬件实现的方案，则会单独建立一个单独的目录“HardwarePRJ”进行存放）
